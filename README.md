Aplicativo realizado bajo los requerimientos pedidos por mi padre para su finca.
Cuenta con seccion de trabajadores, mas que nada para tener sus datos a mano, lotes para administrar los labores de los 4 lotes con los que cuenta la finca.
Y finanzas, el apartado para calcular el salario mensual de cada trabajador en base a su sueldo base, y a sus labores realizadas.
Use Spring Boot para construirlo ya que era lo mejor, debido a ser 3 microservicios.
